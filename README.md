# Italian OTC Medicine (Bozza)

## Overview
This project aims to provide comprehensive information about over-the-counter (OTC) medicines available in Italy. It serves as a resource for consumers to understand the usage, dosage, and potential side effects of various OTC medications.

## Features
- Detailed descriptions of OTC medicines
- Dosage guidelines
- Side effects and warnings
- Search functionality for easy navigation

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/Italian-OTC-Medicine.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Italian-OTC-Medicine
    ```
3. Install the required dependencies:
    ```bash
    npm install
    ```

## Usage
To start the application, run:
```bash
npm start
```
Then open your browser and go to `http://localhost:3000`.
